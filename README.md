# Kubernetes Imperative vs Declarative

This bash script simulates imperative and declarative Kubernetes management using kubectl. It first creates and scales resources with direct commands (imperative) and then defines the desired state in YAML files and applies them (declarative).

## Language

`bash`

## How to Run

Ensure kubectl is installed and configured to a Kubernetes cluster. Run the script using: bash manage_k8s.sh
